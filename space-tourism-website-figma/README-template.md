# Resumo

É um projeto prático utilizando HTML, CSS e JavaScript para construção de um site que fala de uma agência espacial, que mostra quais são os destinos dos astronautas, como lua e marte, as tecnologias utilizadas, e a equipe responsável para a organização e a preparação para as viagens, este prjeto utilizei somente HTML,CSS e JavaScript para praticar os conceitos que aprendi até o momento, sem utilizar framework, praticando conceitos de links, animações, transform CSS, GridLayout, FlexBox e entre outros.
